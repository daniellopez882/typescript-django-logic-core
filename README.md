# typescript-django-logic-core

A high-performance boilerplate for building robust web applications with Django and React, enhanced with the type-safety of TypeScript. This project focuses on clean architecture, scalable patterns, and an elite developer experience.

## Key Features
- **Type-Safe Frontend**: Fully implemented in TypeScript for better reliability and maintenance.
- **Robust Backend**: Powered by Django 5 and Django REST Framework.
- **Enterprise-Ready**: Designed with scalability and performance in mind.

## Project Structure
- `api/`: Backend Django application and REST API logic.
- `frontend/`: React application using TypeScript and modern build tools.
- `core/`: Shared configurations and utilities.

## Setup Instructions
1. **Backend**: Install dependencies via Poetry/Pip and run migrations.
2. **Frontend**: Install dependencies via npm/yarn and start the development server.
